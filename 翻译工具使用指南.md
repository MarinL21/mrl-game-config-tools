# 🌐 多语言翻译工具使用指南

## 📝 快速开始

**使用 Google Gemini AI 自动翻译，质量高、速度快、完全免费！**

### 前置准备：获取免费 API Key（仅需一次）

1. 访问 **[Google AI Studio](https://aistudio.google.com/app/apikey)**
2. 使用 Google 账号登录
3. 点击 **"Create API key"** 创建免费 API Key
4. 复制生成的 API Key（格式：`AIzaSy...`）

📖 **详细教程**：[Gemini API 配置指南](Gemini%20API配置指南.md)

---

## 🚀 开始使用（4步完成）

### 步骤1：配置 API Key

1. 打开翻译工具页面
2. 在顶部 **"🔑 Google Gemini API 配置"** 区域
3. 粘贴您的 API Key
4. 点击 **"🔌 测试连接"** 验证

✅ **一次配置，永久使用**：API Key 会自动保存在浏览器中

### 步骤2：输入中文文本

在输入框中，每行输入一条需要翻译的文本。

**示例：**
```
规则
如何取得点数
注意事项
进行中
```

### 步骤3：AI 自动翻译

点击 **"🤖 AI 翻译"** 按钮，系统会：
- 自动调用 Gemini AI
- 逐行翻译到 17 种语言
- 实时显示翻译进度

⏱️ **翻译速度**：
- 单行：1-2 秒
- 10 行：10-20 秒
- 100 行：2-3 分钟

### 步骤4：复制到 Excel

翻译完成后：
1. 点击 **"📋 复制表格到剪贴板"**
2. 打开 Excel/Google Sheets
3. 粘贴（Ctrl+V 或 Cmd+V）

完成！🎉
